## The Prompt

```
Create a DVD-Rom Style animation of this image https://i.pinimg.com/600x/ac/82/57/ac8257e1cfc4e63f5c63f3d4869eb7c4.jpg
The graphic moves smoothly across the screen in a straight line, bouncing off the edges of the screen whenever it hits a border. Each time it hits a corner or side, it changes direction, creating a continuous floating, ricocheting motion. The speed is steady, the movement is linear, and the object keeps rotating around the screen endlessly, just like the classic DVD logo screensaver.

Change the color on every bounce, no rotation, Make the animation speed fast.
```

## What This Generates

This prompt creates a nostalgic DVD screensaver-style animation with:

- An image (Doge meme) that moves across the screen
- Smooth linear movement at a steady speed
- Bouncing physics when hitting screen edges
- Direction changes upon hitting borders
- Color changes with each bounce
- No rotation (image stays upright)
- Fast-paced movement
- Endless looping animation

## Customization Tips

### Use Your Own Image
Replace the image URL with any publicly accessible image:
```
Create a DVD-Rom Style animation of this image https://example.com/your-logo.png
```

### Adjust Speed
- "Make the animation speed slow" for relaxed movement
- "Use medium speed" for balanced pacing
- "Make it move very fast" for energetic content

### Control Color Changes
- "Change color randomly on every bounce"
- "Cycle through rainbow colors in order"
- "Alternate between red and blue"
- "Don't change colors, keep it white"

### Add or Remove Rotation
- "Add slow rotation as it moves" for spinning effect
- "Rotate 90 degrees on each bounce"
- Default: "no rotation" keeps the image upright

### Modify Bounce Behavior
- "Slow down slightly on each bounce" for damping effect
- "Speed up on corner hits" for dynamic pacing
- "Add a subtle bounce squash effect" for cartoony feel

### Change Background
- "Black background" (classic screensaver style)
- "White background"
- "Gradient background from blue to purple"

## Image Recommendations

<Steps>
  <Step title="Use square or near-square images">
    Images with similar width and height work best for bouncing animations.
  </Step>
  
  <Step title="Ensure public accessibility">
    The image URL must be publicly accessible (not behind authentication).
  </Step>
  
  <Step title="Optimize for clarity">
    Use images that are clear and recognizable even when moving quickly.
  </Step>
  
  <Step title="Consider transparent backgrounds">
    PNG images with transparency create cleaner results than those with solid backgrounds.
  </Step>
</Steps>
